Python Project Generator v1.0
=============================

Creates a new skeleton for your python project

Install
-------

Just put it in your paht or wherever you want, to use it give it execution permisions

Usage
-----

Pass the name of the new project to the script as a parameter, for example:

$> pyprogen skeleton

and the directory skeleton will be generated with the structure: 

generated structure
skeleton
├── bin
├── docs
├── setup.py
├── skeleton
│   └── __init__.py
└── tests
    ├── __init__.py
    └── skeleton_tests.py

4 directories, 4 files

if everything goes ok you'll see the result of the autogenerated test:

testing the project with nosetests
.
----------------------------------------------------------------------
Ran 1 test in 0.007s

OK

The script generate a setup.py file that contains the general information for the project, this information have to be modified to your needs.

NOTE: the files with extension .pyc are generated from the test execution
