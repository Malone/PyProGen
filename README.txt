Python Project Generator v1.0
=============================

Creates a new skeleton for your python project

Install
-------

Just put it in your paht, or wherever you want to useit, give it execution permisions

Usage
-----

just pass the name of the new project to the script, for example:

$> pyprogen skeleton

and it will be generated with the structure below

skeleton/
├── NAME
│   ├── __init__.py
│   └── __init__.pyc
├── bin
├── docs
├── setup.py
└── tests
    ├── NAME_tests.py
    ├── NAME_tests.pyc
    ├── __init__.py
    └── __init__.pyc

4 directories, 7 files

if everything goes ok you'll see the result of the autogenerated test:

testing the project with nosetests
.
----------------------------------------------------------------------
Ran 1 test in 0.009s

OK

The script generate a setup.py file that contains the general information for the project, this information have to be modified to your needs.
